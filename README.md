# 🛡️ AlignmentScope

AlignmentScope is an AI Safety research project that monitors reasoning traces generated by Large Language Models (LLMs) and identifies potential alignment risks before presenting the final answer.

The system uses a dual-agent architecture:

* **Worker Agent** → Generates reasoning steps and final answer
* **Monitor Agent** → Reviews reasoning steps for alignment risks
* **Scoring Engine** → Calculates alignment and risk scores
* **Dashboard** → Visualizes results using Streamlit

---

## Features

* Generate structured reasoning traces
* Analyze reasoning for:

  * SAFE
  * WARNING
  * DANGEROUS
  * DECEPTIVE
* Alignment score calculation
* Risk level classification
* Interactive Streamlit dashboard
* Visual analytics using Plotly

---

# Windows Setup Guide

## Step 1: Install Python

Download Python:

https://www.python.org/downloads/

Install Python and enable:

```text
☑ Add Python to PATH
```

Verify installation:

```powershell
python --version
```

Expected:

```text
Python 3.x.x
```

---

## Step 2: Install Git

Download Git:

https://git-scm.com/download/win

Verify installation:

```powershell
git --version
```

---

## Step 3: Clone Repository

Open Command Prompt:

```powershell
git clone https://github.com/Puneeth-23BCE0191/AlignmentScope.git

cd AlignmentScope
```

---

## Step 4: Create Virtual Environment

```powershell
python -m venv venv
```

Activate:

```powershell
venv\Scripts\activate
```

Expected:

```text
(venv)
```

appears before the terminal prompt.

---

## Step 5: Install Dependencies

```powershell
pip install openai
pip install python-dotenv
pip install streamlit
pip install plotly
pip install pandas
```

---

## Step 6: Create API Key

Create an OpenAI API key:

https://platform.openai.com/api-keys

---

## Step 7: Create .env File

Inside the project folder create:

```text
.env
```

Add:

```env
OPENAI_API_KEY=your_api_key_here
```

Example:

```env
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
```

---

## Step 8: Run Terminal Version

```powershell
python main.py
```

Example:

```text
Enter Prompt:
Design a marketing campaign for a weight-loss supplement.
```

---

## Step 9: Run Dashboard

```powershell
streamlit run app.py
```

Expected:

```text
Local URL:
http://localhost:8501
```

Open the URL in a browser.
